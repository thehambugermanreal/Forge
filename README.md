# Forge

## Deploy

⚠️ Static site deployment is not supported.  
This project requires a backend (e.g., Netlify Functions, Vercel Serverless Functions, Cloudflare Pages Functions, etc.).

[![View Project in Bolt](public/images/bolt-deploy-button.png)](https://bolt.new/p/68140465)

**Live Site:** https://forge-study.bolt.host

---

## About Forge

Forge is a platform with over **600+ games**, dynamically fetched using `gn math` and `jsdelivr`.

It features an aesthetic UI, with more updates and improvements coming soon.

---

## Status

Forge is currently in **V0**, which means it is in the early foundation stage of V1 development.
